# Wochenanalyse & Projektideen-Generator - Quick Start Guide

## 5-Minuten Setup

### 1. Aufnahmen erstellen
```
1. Öffnen Sie die Voice Intelligence App
2. Klicken Sie auf "Start Recording"
3. Sprechen Sie Ihre Gedanken, Pläne und Ideen auf
4. Speichern Sie mehrere Aufnahmen während der Woche
```

### 2. Zur Analytics Seite navigieren
```
1. Klicken Sie auf "Analytics" in der Sidebar (TrendingUp Icon)
2. Sie sehen das Analytics Dashboard
```

### 3. Wochenanalyse generieren
```
1. Klicken Sie auf "Generate Analysis" Button
2. Das System analysiert Ihre Aufnahmen
3. Sie erhalten eine Zusammenfassung mit Insights
```

### 4. Projektideen anschauen
```
1. Klicken Sie auf "Generate Ideas" Button
2. Das System generiert Projektvorschläge
3. Jede Idee hat Effort-Level und Impact-Rating
```

---

## Wichtige Features erklärt

### Summary Card
Zeigt die Zusammenfassung Ihrer Woche:
- **Zusammenfassung:** Hauptthemen und Fokusbereich
- **Aufnahmen:** Anzahl der Aufnahmen diese Woche
- **Tasks:** Abgeschlossene Aufgaben

### Metrics Grid
Drei wichtige KPIs:
- **Total Recordings:** Wie viele Aufnahmen diese Woche
- **Top Theme:** Das am häufigsten diskutierte Thema
- **Productivity Trend:** Trend im Vergleich zur Vorwoche

### Charts
- **Recording Trends:** Zeigt Aufnahmen über 4 Wochen
- **Top Themes:** Pie Chart der Diskussionsthemen

### Project Ideas
Intelligente Projektvorschläge mit:
- **Effort Level:** Low/Medium/High (Aufwand)
- **Impact:** Low/Medium/High (Potenzial)
- **Skills:** Benötigte Fähigkeiten
- **Timeline:** Geschätzter Zeitrahmen

## Häufig gestellte Fragen

**F: Wie oft sollte ich Analysen generieren?**
A: Wöchentlich ist ideal. Das System analysiert automatisch die Aufnahmen der aktuellen Woche.

**F: Brauche ich viele Aufnahmen für gute Analysen?**
A: 3-5 qualitativ hochwertige Aufnahmen pro Woche sind ausreichend. Mehr Aufnahmen = bessere Analysen.

**F: Kann ich Analysen exportieren?**
A: Derzeit können Sie Screenshots machen. Ein Export-Feature ist geplant.

**F: Sind die Projektideen realistisch?**
A: Das LLM basiert auf Ihren Transkriptionen. Je detaillierter Ihre Aufnahmen, desto besser die Ideen.

**F: Kann ich Analysen von vergangenen Wochen anschauen?**
A: Ja, die Trends zeigen die letzten 4 Wochen. Ältere Analysen werden in der Datenbank gespeichert.

## Tipps für beste Ergebnisse

### 1. Qualität der Aufnahmen
- Sprechen Sie klar und deutlich
- Vermeiden Sie Hintergrundgeräusche
- Nutzen Sie Sätze statt einzelner Wörter

### 2. Regelmäßigkeit
- Nehmen Sie mehrmals pro Woche auf
- Konsistenz führt zu besseren Trends
- Wöchentliche Analysen sind optimal

### 3. Detailreichtum
- Erklären Sie Ihre Gedanken und Pläne
- Nennen Sie konkrete Projekte und Ziele
- Beschreiben Sie Herausforderungen und Lösungen

### 4. Feedback Loop
- Nutzen Sie Projektideen als Inspiration
- Verfolgen Sie Ihre Fortschritte
- Passen Sie Ihre Pläne basierend auf Insights an

## Fehlerbehandlung

### "No recordings found for this week"
**Lösung:** Erstellen Sie mindestens eine Aufnahme diese Woche

### "LLM API not available"
**Lösung:** Überprüfen Sie Ihre Internetverbindung oder nutzen Sie Offline-Mode

### Charts werden nicht angezeigt
**Lösung:** Aktualisieren Sie die Seite (F5) oder leeren Sie den Browser-Cache
